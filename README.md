# Signy

A real-time hand gesture recognition system that translates hand gestures captured via webcam into text or synthesized speech. Built with MediaPipe, scikit-learn, OpenCV, and Streamlit.

---

##  Features

- **Real-time hand tracking** using MediaPipe Hands
- **Gesture-to-text** mode: compose sentences letter by letter using gestures
- **Gesture-to-speech** mode: instantly speak gestures or queue a full sentence
- **Tkinter GUI dashboard**: combined text/speech modes with live metrics panel
- **Streamlit web dashboard**: browser-based launcher, live snapshot inference, and model metrics
- **Data collection pipeline**: guided capture tool with countdown and progress bar
- **Robust model training**: Random Forest + Gradient Boosting ensemble with anti-overfitting splits
- **Model evaluation**: per-class F1, weak-class detection, confusion matrix, and live FPS profiling

## Quick Start

### 1. Install dependencies

```bash
pip install -r requirements.txt
```

On Linux, also install the system packages listed in `packages.txt`:

```bash
sudo apt-get install libgl1 libglib2.0-0 libsm6 libxext6 libxrender1
```

### 2. Collect gesture data

```bash
python capture_data.py
```

Follow the on-screen prompts. For each gesture, press **Space** to begin recording 150 samples. Repeat for all 10 built-in gestures.

### 3. Train the model

```bash
python train_model.py
```

This runs a full training pipeline: data exploration → ordered train/val/test split → cross-validated model selection (RF, GB, and RF+GB ensemble) → saves `models/gesture_model.pkl` and `models/label_encoder.pkl`.

### 4. Evaluate the model

```bash
# Offline evaluation only (no webcam required)
python test.py --no-live

# Full evaluation including live FPS profiling
python test.py --duration 20
```

### 5. Run an application

```bash
# Gesture → text (OpenCV window)
python app1_text.py

# Gesture → speech (OpenCV window)
python app2_speech.py

# Full Tkinter GUI dashboard
python dashboard.py

# Streamlit web dashboard
streamlit run streamlit_app.py
```

## Supported Gestures

| Gesture | Default action |
|---------|---------------|
| `hello` | Appends "hello" to the buffer |
| `yes` | Appends "yes" |
| `no` | Appends "no" |
| `call` | Appends "call" |
| `iloveyou` | Appends "iloveyou" |
| `four` | Appends "four" |
| `ok` | **CONFIRM** — speaks the queued sentence (speech mode) |
| `peace` ✌️ | **SPACE** — inserts a space |
| `stop` ✊ | **BACKSPACE** — deletes the last character |
| `point` ☝️ | **CLEAR** — clears the entire buffer |

Special gesture mappings are configurable via `SPECIAL_GESTURES` at the top of each app file.

---

## Configuration

Each app file exposes constants at the top for easy tuning:

| Constant | Default | Description |
|----------|---------|-------------|
| `CONFIDENCE_THR` | `0.90` (text) / `0.50` (speech) | Minimum model confidence to accept a prediction |
| `COOLDOWN_SEC` | `1.5` (text) / `1.2` (speech) | Minimum seconds between accepted gestures |
| `TTS_ENGINE` | `"winsapi"` | TTS backend: `"winsapi"`, `"pyttsx3"`, or `"gtts"` |
| `TTS_LANG` | `"en"` | Language code for gTTS |
| `SPEECH_RATE` | `150` | Words-per-minute for pyttsx3 |
| `SPEECH_MODE` | `"word"` | `"word"` (speak on CONFIRM) or `"instant"` (speak each gesture) |
